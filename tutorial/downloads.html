Marquees Presets
A set of common banner size selection, useful for spritesheet selections, cropping from screenshots.
Instructions     Download
Loading Selections template (You only have to do it once)

Download “Marquees_Presets.tpl".

Go to "Marquee Tool” -> Load Tools Preset

After loading of the preset:


Click anywhere on the arboard, you will have a marquee in exact banner size.
