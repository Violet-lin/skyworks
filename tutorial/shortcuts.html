Keyboard Shortcuts

Searching for the tools you want from top menu bar?
Keyboard shortcut might save you some time on frequent-use command.
  • Group layers. cmd+G
    cmd+G

    Select the layers that you want to group, and cmd+G.


  • photo_size_select_small Select all. cmd+A
    cmd+A

    Create a selection marquee by cmd+A.

  • photo_size_select_small Deselect all. cmd+D
    cmd+D

    Removing all active marquee selections.

    Especially when you realised brush, fill or gradient tools are not working.

  • Free Transform. cmd+T
    cmd+T

    Free transform and resize. It is also very useful to get values like x, y, width, height for DCO and Bespoke banners. You can set to read from different reference point, regX/regY. (E.g top-left or center-middle)

  • format_color_fill Fill background background. cmd+backspace
    cmd+backspace

    Fill background background.

  • format_color_fill Fill foreground background. option+backspace
    option+backspace

    Fill foreground background.

  • palette Swap Foreground / Background colour. X
    X

    Swap Foreground / Background colour.

  • palette Default colours. D
    D

    Default colours.
    Foreground: Black / Background: White.

  • Toggle lock and unlock on selected layer(s). cmd+/
    cmd+/

    Select layers you would like to lock and unlock, and press cmd+/.

  • Move selection 1 pixel. Arrow keys
    Arrow keys

    Move selection 1 pixel

  • Move selection 10 pixels. shift + arrow keys
    shift + arrow keys

    For every shift + arrow keys, selection(s) are moved 10 pixel.
    You don't have to count anymore when moving 320 to 300.

  • Save for Web. cmd+option+shift+s
    cmd+option+shift+s

    Save for Web allows you to select file formats to export (jpg, png and gif),
    customised transparency, quality, file size and etc.

  • Duplicate selections. cmd+J
    cmd+J

    Duplicate selections. Create a new layer from selected layer(s) or selected marquee area(s)

  • Cut selections, and create a new layer. cmd+shift+J
    cmd+shift+J

    Similar as cmd+J, just that it cut selections.
    Create a new layer from selected layer(s) or selected marquee area(s)
